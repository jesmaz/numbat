6.3.1: Initialisation constructor

Initialisation constructors come in to flavours. With arguments and without.
```
#No arguments
def string () -> (string ret) {
	#code
}

#One argument
def string (int64 num) -> (string ret) {
	#code
}

#Two arguments
def string (int64 num, double d) -> (string ret) {
	#code
}
#And so fourth
```
